# PCB Test Manager

A web-based application for managing PCB (Printed Circuit Board) test projects.

## Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Running the App

```bash
run.bat
```

Or directly:

```bash
python app.py
```

The app will be available at `http://localhost:5000`.
